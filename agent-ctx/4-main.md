# Task 4: Major overhaul of daily-challenge-screen.tsx

## Files Modified
- `src/components/screens/daily-challenge-screen.tsx` — Complete rewrite with 12 new features
- `src/lib/game/player-progression.ts` — Added barrel exports for Challenge Mixer

## Changes Summary

### 1. Profile Selector at Top
Added clickable chip row between title and Level card. Shows all active profiles with avatar/name. Clicking switches the active profile.

### 2. Three Song Choices
Replaced the single "Play Now" random song button with a 3-song selection grid. Songs are shuffled on mount and when `completedToday` changes. Each card shows title, artist, and duration.

### 3. Best Result Display
When the challenge isn't completed yet, shows the player's best attempt today from `getPlayerBestResult()` with a progress bar and percentage toward the target.

### 4. Leaderboard Correct Metric
Leaderboard now displays the metric matching the challenge type (score/accuracy/combo/perfect notes) instead of always showing raw score.

### 5. Weekly Challenge Tab
Added 5th tab "Weekly" between "Challenges" and "Challenge Modes". Shows weekly challenge description, scaled target for player level, and time until Monday reset.

### 6. Quests Section
Added quest list below the badges section in the Badges tab. Each quest shows icon, name, description, progress bar, and a "Claim" button when completed but not yet claimed.

### 7. Challenge Mixer
Added custom challenge builder in Modes tab. Shows 8 available modifiers as toggleable buttons. When modifiers are selected, shows difficulty label and XP calculation. Play button creates a custom challenge and starts the game.

### 8. Weekly Progress Calendar
Added Mon-Sun progress dots inside the XP progress card. Green check marks for completed days, gray dots for incomplete.

### 9. Removed 'songs' Challenge Type
Removed the `songs` entry from `challengeDescriptions` since it's never generated by the daily challenge system.

### 10. Co-op Mode
Added green "Co-op" toggle button alongside Single/Duel in the game mode selector.

### 11. Dynamic Difficulty Indicator
Shows a note below the challenge description when the player's level is > 1, explaining that the target is scaled (shows base target and scaled target).

### 12. Challenge Mode Song Selection
Modes tab now shows 3 song choices when a mode card is clicked (instead of immediately playing a random song). Clicking the same mode again deselects it.
